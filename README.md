# Customer Engagement & Churn Drivers Analysis

## Question
Which customer behaviors and characteristics are most strongly associated with churn, and what patterns emerge across customer segments?

## Data
This project uses a synthetic customer churn dataset simulating user demographics, account characteristics, and engagement-related features for a subscription-based service.

## Approach
- Data cleaning and validation
- Exploratory data analysis
- Segment-level churn analysis
- Visualization of key patterns
- (Optional) Simple modeling to validate findings

## Status
Scaffolding and initial exploration
